## Description
This Android application demonstrates basic object-oriented programming concepts.  
The project implements two classes, Lamp and Bulb, which simulate the behavior of a lamp with adjustable light intensity and bulb replacement functionality.  
A simple test mechanism verifies the required scenarios and displays results in Logcat.
